Đối với app MyApp/config/database/databaseCon.js
Đối với phần Website admin Server/src/resource/views 

Để chạy được trước tiên tải ứng dụng Expo Go hoặc mở máy ảo.
Bước 1: sử dụng terminal nhập: npm install để tải các gói phụ thuộc. Lặp lại việc này với cả 2 thư mục.
Bước 2: sau đó ở thư mục Server nhập: npm start, 
Bước 3: sau khi chạy server xong thì di chuyển qua thu mục MyApp nhập: npm run android, nó sẽ hiển thị QR và bạn sẽ quét thông qua app Expo go.
